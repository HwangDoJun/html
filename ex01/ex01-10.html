웹 표준

웹표준이란 브라우저 종류 및 버전에 따른 기능 차이에 대하여 호환이 가능하도록 제시된 기준이며, 서로 다른 기술이나 버전이 구현되는 것을 동일하게 구현함은 물론 어느 한 족으로 치우치지 않게 공통요소를 사용하여 웹 페이지를 제작하는 기술이다.



웹 표준의 장점

1. 수정 및 운영관리가 용이함: 콘텐츠의 올바른 구조화와 CSS로 시각표현을 통일하게 제어함으로써 페이지 제작의 부담을 감소시키고 관리하기 용이하게 해줌.

2. 접근성 향상: 웹 표준으로 작성한 문서의 경우 다양한 브라우징 환경에 대응이 가능하므로 폰, PDA등에서도 구현이 되므로 접근성에 있어서 유용한 편이다.

3. 검색엔진 최적화: 구조화된 웹 페이지는 검색 로봇 수집을 통해 검색엔진에 효율적으로 노출될 수 있도록 검색엔진의 검색결과를 최적화.

4. Fire size 축소 및 서버 저장 공간 절약: 효율적인 소스 작성은 파일 사이즈와 저장 공간을 절약할 수 있으며 화면표시에 소요되는 시간 단축에도 용이함.

5. 효율적인 마크업: CSS와 HTML을 분리하여 제작하기 때문에 불필요한 마크업이 최소화되어 페이지 로딩속도가 향상됨.

6. 호환성 가능: 기존 IE브라우저에서만 작성이 가능했던 요소들이 웹 표준을 준수함으로써 다양한 브라우저(크롬, 파이어폭스, 사파리, 오페라)에서도 작동 가능.
