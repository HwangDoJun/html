CSS 관련 리포트


CSS: Cascading Style Street..의 약자이며, 사용자에게 문서를 표시하는 방법을 지정하는 언어입니다(ex. 스타일, 레이아웃 등등)

문서는 마크업 언어를 사용하여 작성하는 텍스트 문서이며, 가장 일반적으로 사용하는 언어로는 HTML이 있으며, 그 외에도 몇몇 마크업 언어들이 존재합니다. 사용자에게 문서를 제공한다는 것은 대상이 사용 가능한 형식으로 문서를 변환하는 것을 의미하며, 주로 모니터, 프린터 등에 나오는 문서를 시각화하도록 설계되었습니다.



CSS 구문

CSS는 규칙 기반 언어이며, 웹 페이지에 있는 특정 요소or요소 그룹에 적용할 스타일 그룹을 지정하는 규칙을 정의합니다.(홈페이지에 관련 예시가 나와있으니 시간날때 참고(코드에 사이트가 존재))

CSS속성의 경우 어떠한 속성을 정했느냐에 따라 값이 달라지기도 합니다. color라는 속성을 통해 다양한 색상 속성을 정할 수 있으며, front size라는 속성을 통해 다양한 크기 단위 값을 사용하는게 가능합니다.

CSS Modules

CSS를 사용하여 스타일을 지정할 수 있는것이 너무 많기 때문에, 관련 언어는 module로 구분하기도 합니다.
